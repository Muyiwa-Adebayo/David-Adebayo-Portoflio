Enthronement Assembly — Locator App

Enthronement Assembly needed a streamlined way for members to discover and join nearby 'Love Circle' fellowship centers. The coordination overhead for administrators was extremely high, and users struggled to find locations based on their exact geographical coordinates.

I built a robust full-stack web application implementing custom geolocation algorithms. The platform features a public-facing discovery app and a secure admin panel with role-based authentication and real-time request tracking, designed for maximum scalability.

Over 50+ centers successfully onboarded in the first phase. The platform reduced manual admin overhead by 80%. Users can now seamlessly find and request to join the nearest center in under 3 clicks, significantly boosting community engagement.

ARGUE — Report Generation Tool

Automated report tools often suffer from low user engagement due to confusing interfaces and lack of visual clarity. The goal was to transform complex data publication into a user-centered experience.

I conducted extensive user research and usability testing to identify pain points. By designing high-quality illustrations and comprehensive design systems, I ensured consistency and scalability across the platform's report generation modules.

The redesign directly enhanced user retention by 30%. Interactive prototypes allowed stakeholders to visualize project goals clearly, resulting in high implementation fidelity and a significantly improved user brand sentiment.
